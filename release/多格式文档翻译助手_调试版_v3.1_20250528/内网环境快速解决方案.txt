========================================
🌐 内网环境快速解决方案
========================================

📋 问题描述：
在内网环境中运行程序时出现连接错误：
- 智谱AI连接失败 (ConnectionAbortedError 10053)
- Ollama服务未运行 (WinError 10061)
- 硅基流动连接问题

🚀 快速解决方案：

方案一：使用内网模式启动器（最简单）
----------------------------------------
1. 双击运行 "内网模式启动器.bat"
2. 程序会自动设置内网模式环境变量
3. 跳过所有外部API连接检查

方案二：手动配置（推荐）
----------------------------------------
1. 运行 "启用内网模式.bat"
2. 选择 Y 确认启用内网模式
3. 配置文件会自动更新
4. 重新启动程序

方案三：环境变量设置
----------------------------------------
在启动程序前执行：
set INTRANET_MODE=true

或在PowerShell中：
$env:INTRANET_MODE = "true"

📋 验证配置：
运行 "python test_intranet_mode.py" 验证配置是否正确

🔧 恢复外网模式：
运行 "恢复外网模式.bat" 即可恢复正常模式

⚠️ 注意事项：
- 内网模式下某些在线翻译功能可能不可用
- 建议安装本地Ollama服务作为翻译引擎
- 程序启动会更快（跳过网络检查）

📞 技术支持：
如有问题请查看 "内网环境解决方案.md" 详细文档
